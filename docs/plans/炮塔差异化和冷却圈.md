# 炮塔差异化重设计 + 冷却读条圈

## 一、三炮塔差异化重设计

### Blaster 冲击炮（#f87171）— "重型单管炮台"

- **基座**：圆形 → **方形** `roundRect`，工业/军事风
- **炮管**：加粗，末端 **V 形炮口**
- **储电**：两个弹药仓方块，分列基座两侧，充满发光
- **开火特效**：
  - 后坐力：约 100ms 内炮管缩短（计划版为 15%；实现若不同以代码为准）
  - 枪口闪光：约 120ms 内黄白色扩散圆环

### Gatling 加特林（#f59e0b）— "旋转多管速射炮"

- **基座**：单圈 → **双圈**（内圈 + 厚外环）
- **炮管**：多管、间距与线宽加大；**旋转动画**（`now * heat * spinSpeed`）
- **储电**：外环上 **5 段弧光**（对应 maxPower）
- **开火特效**：
  - 管口火花：约 80ms 内 2–3 条随机短线
  - 弹壳粒子：每发微粒（复用 `state.particles`）

### Sniper 狙击手（#a78bfa）— "精密轨道炮"

- **基座**：圆形 → **菱形**（45° 旋转方框）
- **炮管**：加长，中段 **稳定翼**，瞄准镜沿管身布置
- **储电**：侧边充能条或四角三角块等形式（与代码实现一致）
- **开火特效**：
  - 炮口光爆：约 200ms 内紫色扩散光球
  - 弹道残影：约 300ms 内沿射击方向渐隐紫色光线

## 二、冷却读条圈（Sniper + Tesla）

### 设计目标

将 Sniper 的下置读条和 Tesla 缺失的冷却指示，统一替换为 **内轮廓弧形读条圈**。

### 视觉设计

- **形态**：沿炮塔内轮廓（inset 矩形或菱形边缘内侧）画弧线，从 12 点钟方向顺时针扫过
- **进度**：`elapsed / cooldown` → 弧线从 0° 扫到 360°
- **颜色**：白色/银色半透明（如 `rgba(255,255,255,0.4→0.7)`），与主题色储电区分
- **线宽**：约 2px，略粗于轮廓 stroke
- **就绪闪光**：扫满后短暂全圈亮闪再消失（可选）

### 与储电显示的视觉区分


| 维度   | 储电显示      | 冷却读条圈    |
| ---- | --------- | -------- |
| 含义   | 当前弹药/能量储备 | 距下次开火的时间 |
| 形态   | 离散方块/弧段   | 连续弧线扫描   |
| 颜色   | 炮塔主题色     | 白色/银色半透明 |
| 位置   | 靠近中心      | 紧贴内轮廓边缘  |
| 存在时机 | 有电时常显     | 仅 CD 中可见 |


### 适用炮塔

- **Sniper**（4000ms CD）：沿菱形内边缘画弧，替代原下置条形读条
- **Tesla**（3000ms CD）：沿圆形内边缘画弧
- **Blaster**：CD 短，一般不画读条圈
- **Gatling**：无传统 CD，保留热量条

## 三、技术实现要点

### 无需新增状态字段（原则上）

- 冷却进度 = `(now - tower.lastActionTime) / cooldownMs`
- 加特林旋转 = `now * heat * SPIN_SPEED`（渲染驱动）
- 开火瞬时效果 = `now - tower.lastActionTime` + 固定时长

### 主要文件

- `src/game/renderer.ts`：`drawTowerDetails` 各炮塔分支、冷却圈、删除 Sniper 下置条
- `src/game/useGameLoop.ts`：开火处粒子等

## 四、差异对比速览

- **基座**：Blaster=方形 / Gatling=双圈 / Sniper=菱形
- **炮管**：Blaster=粗管+V口 / Gatling=多管旋转 / Sniper=长管+稳定翼
- **储电**：Blaster=弹药格 / Gatling=5 段外弧 / Sniper=充能条或角块
- **开火**：Blaster=后坐力+闪光 / Gatling=火花+弹壳 / Sniper=光爆+残影
- **冷却**：Sniper + Tesla=白色内轮廓读条圈；Gatling=热量条

# 规划文档（Plan）

本目录集中存放由 Cursor Plan 生成或整理的设计/规划 Markdown，便于版本管理与检索。

**说明：** Cursor 在本地可能另有副本（例如用户目录下的 `.cursor/plans/`）；此处为仓库内的权威副本，新增计划时请保存到此目录。
